# PopMate（ポップメイト）

プライスポップ作成サービス - スマレジAPI連携

## 概要

ポップメイトは、スマレジAPIから商品情報を取得し、プライスポップを自動で作成するWebサービスです。手動でのデータ入力とデザイン作成の手間を削減し、誰でも簡単かつ迅速に質の高いポップを作成できます。

## システム構成

| コンポーネント | サービス | 役割 |
| :--- | :--- | :--- |
| フロントエンド | Vercel | ユーザーインターフェース |
| バックエンド | Railway | API処理・ビジネスロジック |
| データベース | Supabase | データ永続化 |
| バージョン管理 | GitHub | ソースコード管理 |

## 主な機能

1. **メイン画面**: テンプレート選択・保存データ呼び出し
2. **テンプレートデザイン**: 自由なポップデザイン編集（0.1mm単位）
3. **データ選択**: スマレジAPIからの商品情報取得・検索
4. **商品情報編集**: リアルタイムプレビューでの編集
5. **印刷機能**: A4自動配置・PDF出力・データ保存
